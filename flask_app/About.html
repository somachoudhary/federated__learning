About the Project

Privacy-Preserving Stress Detection

This project leverages Federated Learning (FL) to detect stress from health data while preserving user privacy. Using the MMASH dataset, we process accelerometer and heart rate data (Axis1, Axis2, Axis3, Vector Magnitude, RR Interval) to predict stress levels without centralizing sensitive data.

Why Federated Learning?

FL enables model training across multiple devices (e.g., 22 clients) without sharing raw data, addressing privacy concerns in healthcare. Our model, built with Flower AI, aggregates weights from clients to create a global model, achieving ~70–90% accuracy.

samAI Hackathon

Developed for the samAI Hackathon (AI for Health & Wellbeing), this project showcases early diagnostics and privacy-preserving AI. The Flask web app provides an interactive interface for real-time stress predictions.
